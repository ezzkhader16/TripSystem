# TripSystem – Android Travel Planner App

TripSystem is a modern Android application designed to help users manage and organize their trips efficiently.  
This project was built as part of an Android Development course using **Java** and **XML**.

---

##  Features

###  Trip Management
- Add new trips with:
  - Trip title  
  - Date  
  - Budget  
  - Trip type (Tourism / Food / Shopping)  
  - Premium Trip option  
  - Payment status (Included / Pay on Arrival)  
  - Place image selection (Jerusalem / South Lebanon / Beirut)

###  Image Support
- Each trip displays a main image on the home screen  
- A secondary image appears in the trip details screen (jer2 / south2 / rawshe2)

###  Trip Details Screen
- Full trip description (English)  
- Extras section (Breakfast / Bike Rental)  
- Auto–calculated total price  
- Edit button (Floating Action Button)

###  Data Persistence
- All trips are saved using **SharedPreferences**  
- Data remains available even after app restart

###  UI / UX
- Professional gradient background  
- Rounded cards with a clean layout  
- Custom drawable components  
- Status badges (Premium Trip • Included / Pay on Arrival)

###  Developer Tools
- Full Activity **Lifecycle logs** in Logcat  
- Modular code structure (Activities + Adapter + Model class)

---

##  Tech Stack
- **Java (Android)**  
- **XML Layouts**  
- **RecyclerView + Custom Adapter**  
- **CardView UI Components**  
- **SharedPreferences (Local Storage)**  
- **Drawable Custom Shapes (UI/UX)**
